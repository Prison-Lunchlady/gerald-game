================================================
  WHAT SHOULD GERALD DO?
  Setup Instructions
================================================

STEP 1 - Install Node.js (one time only)
-----------------------------------------
Node.js is the engine that runs the game.
You only install this once, ever.

Go to this website and download the LTS version:

  https://nodejs.org

Click the big green button that says "LTS"
Run the installer and click Next through everything.
Don't change any settings.


STEP 2 - Extract this zip file
-----------------------------------------
If you haven't already, right-click the zip file
and choose "Extract All", then click Extract.


STEP 3 - Launch the game
-----------------------------------------
Go into the "What should gerald do" folder.
Double-click the file called:

  PLAY_GERALD.bat

It will do everything else automatically and
open the game in your browser.


STEP 4 - Play
-----------------------------------------
The game opens at:  http://localhost:5173

Controls:
  SPACE or BOB button   = Bob Gerald upward
  LEFT / RIGHT arrows   = Move Gerald
  A / D keys            = Also move Gerald

Goal: Get Gerald to the far right side of the pool
without letting his DROWN meter hit 100%.

Collect bubbles to reduce the DROWN meter.
Collect gold coins to earn Gerald Points.
Spend Gerald Points in the upgrade shop.


EVERY TIME AFTER THAT
-----------------------------------------
Just double-click PLAY_GERALD.bat.
Node.js is already installed, so it goes
straight to the game.


NEED HELP?
-----------------------------------------
If the bat file shows a red error, it means
Node.js is not installed yet. Go back to Step 1.
